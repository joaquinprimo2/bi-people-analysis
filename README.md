# BI People Analysis – Data Analyst Project

## Overview

This project analyzes a structured dataset of people to extract meaningful business insights using SQL and Power BI.

The objective is to demonstrate core Business Intelligence skills including data exploration, aggregation, segmentation, and dashboard visualization.

This project focuses on descriptive analytics and insight generation.

## Dataset

The dataset contains structured information about individuals:

- Name
- - Age
  - - City
    - - Profession
     
      - The data is used for educational and portfolio purposes.
     
      - ## Objectives
     
      - The analysis aims to:
     
      - - Understand population distribution by city
        - - Identify profession distribution patterns
          - - Calculate average age by city and profession
            - - Generate clear insights to support data-driven decisions
             
              - ## Analysis Performed
             
              - ### SQL Analysis
              - - Count of people by city
                - - Count of people by profession
                  - - Distribution by city and profession
                    - - Average age by profession
                     
                      - ### Power BI
                      - Interactive dashboard including:
                      - - People by city
                        - - People by profession
                          - - Average age by city
                            - - Age segmentation
                             
                              - ## Key Insights
                             
                              - - Certain cities concentrate higher volumes of specific professions.
                                - - Profession distribution reveals dominant population segments.
                                  - - Average age varies depending on city and profession, suggesting demographic differences.
                                    - - Basic BI techniques can extract actionable insights without predictive modeling.
                                     
                                      - ## Tools & Technologies
                                     
                                      - - SQL (SQLite)
                                        - - Power BI
                                          - - DB Browser for SQLite
                                            - - Visual Studio Code
